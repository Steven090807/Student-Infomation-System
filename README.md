## 📌 Project Overview
The **Student Information System (SIS)** is a full-stack application built to simplify administrative tasks within educational institutions. It provides a centralized platform for managing student details, course enrollments, and academic performance, replacing traditional manual record-keeping with a secure, real-time digital solution.
<br>

## 🛠️ Technical Stack

| Layer | Technologies |
| :--- | :--- |
| **Frontend** | React.js (Vite), Material UI (MUI), PrimeReact |
| **Backend** | Node.js (Express) |
| **Database** | MySQL |
| **UI & Libraries** | Formik, Yup, Axios, Prime React |
<br>


## Key Features

- **Teacher Portal:** Manage class assignments and student grades.
- **Student View:** Real-time access to grades and teacher information.
- **Submission Tracking:** Clear visualization of **"Pending"** vs. **"Submitted"** tasks.
